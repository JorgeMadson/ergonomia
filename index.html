A História da Web


A Internet foi criada a partir da Guerra Fria em 1969, mais especificamente 07/04/1969, com a ARPANET. Porém, alguns anos antes já se era falada da criação de uma “Rede Intergalática de Computadores”.
O termo “Internet” só foi criado nos anos 70, com a primeira descrição de protocolos TCP/IP, e em 1988 os civis tiveram a oportunidade de conhecer essa ferramenta.
Em 1992, o cientista Tim Berners-Lee criou a World Wide Web (o famoso www) e, para efetivar o nascimento da internet que temos hoje, foi criado o protocolo HTTPS, que garante o envio de criptografias pela rede.
A partir de então, a internet se popularizou a um ponto que hoje é extremamente difícil conhecer alguém que não tem uma conexão banda-larga em casa ou um celular com 3-4G.
O que vem pela frente? A tendência é que a internet se espalhe ainda mais e fique cada vez mais portátil, possibilitando que o usuário use o máximo da ferramenta.

HTML Básico

Para começar, o HTML não é uma linguagem de programação. Ela consiste em elementos, que unidos criam uma página web do jeito que o usuário deseja.

Importante saber!

Toda linha deve ser finalizada com a barra antes do elemento (letra). Por exemplo:

Um parágrafo é identificado com os códigos <p> e </p>.

<strong> </strong> é utilizado para negritar uma frase/palavra.

Exemplo HTML

Assim para um arquivo html ser lido pelo navegador algumas tags são mínimas são:

							 
	<!DOCTYPE html>
	<meta charset="UTF-8">
	<html>
	<body>
	
	<h1>Meu primeiro cabeçalho</h1>
	<p>Meu primeiro parágrafo.</p>
	
	</body>
	</html>

CSS Iniciante

Códigos CSS são códigos para estilizar a página HTML que é desenvolvida. Caso o desenvolvedor queira colocar linhas vermelhas, por exemplo, é possível editar com o CSS. Como? Existem dois métodos:
  1. Fazendo o código em um arquivo a parte, salvando o mesmo como *.css na pasta “styles”. Após isso, você deve adicionar o código de referência entre as tags e , exemplo: 
    <link href="styles/style.css" rel="stylesheet" type="text/css">
  2. Fazendo no corpo do arquivo HTML. 
    
<!DOCTYPE html>
<html>
<head>
	<style>
	body {
		background-color: lightblue;}
	</style>
</head>
</html>

  3. O conteúdo de um código CSS é contido por: Seletor (nome do elemento HTML a ser estilizado) Declaração (para especificar as propridades para estilizar) Propriedades (forma que você quer estilizar o HTML) Valor da Propridade (o que você quer mudar dentro da propriedade)

    NÃO SE ESQUEÇA!

    • Linha de comando necessita de chaves.
    • : (dois pontos) para separar propridades do valor da propriedade.
    • ; (Ponto e vírgula) para separar as declarações.

Exemplo CSS

				
<!DOCTYPE html>
<html>
<head>
	<style>
body {
	background-color: lightblue;
}
h1 {
	color: white;
	text-align: center;
}
p {
	font-family: verdana;
	font-size: 20px;
}
</style>
</head>
<body>
	<h1>Meu Primeiro CSS</h1>
	<p>Isso é um paragráfo.</p>
</body>
</html>

Javascript Básico

Javascript é uma linguagem de programação que permite adicionar interações (animações, efeitos) na página WEB desenvolvida via HTML. Para desenvolver em javascript, assim como o CSS, existem dois métodos:

  1. Primeiramente é necessário criar um arquivo chamado “main.js” e salvar na pasta “scripts” Após isso, deve-se fazer exatamente como no CSS, chamar o arquivo dentro do HTML. Exemplo: 
    							
<script src="scripts/main.js"></script>
  2. Fazendo no corpo do arquivo HTML 
    							
<button type="button" onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'>Click Me!</button>
  3. Variáveis são locais para armazenar valores. Para chamar a mesma, você declara a palavra-chave var. As variáveis mais importantes são: String (texto), Number (números), Boolean (true/false), Array (armazena vários valores numa mesma referência).

Javascript Exemplo

				
<!DOCTYPE html>
<html>
<body>

<h2>O que JavaScript pode fazer?</h2>

<p id="demo">JavaScript pode mudar conteúdo HTML.</p>

<button type="button" onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'>Click Me!</button>

</body>
</html>

Dicas

1. Sempre tente usar ferramentas que facilitam o desenvolvimento, como o VS Code que é uma programa em que você pode desenvolver seus códigos e ele tem funcionalidades como autocompletar as tags HTML para que você não esqueça de fechar o Body da página, como exemplo, isso facilita muito.

2. Leia o código de outras pessoas, o programador passa a maior parte do tempo lendo código ao invés de escrever, por isso a leitura e intrepretação de código é muito importante .
